## Steps to setup
Make sure in backend folder

```bash
1. npm init -y
2. npm install firebase-admin
3. npm install express cors dotenv
4. npm install -D nodemon
```


### To start:
```bash
npm start
```
### File corrupt? To check:
```bash
Diagonose: node diagnose.js
```

### Seed data (dummy data):
```bash
node seedDatabase.js //(already in database right now)
```
to delete

```bash
node seedDatabase.js --clear
```
